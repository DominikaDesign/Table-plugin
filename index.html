Dynamic table plugin

Create your own table in few easy steps!

Getting Started

The only thing is to link css plik in your <head> section. 

                
  <link rel="stylesheet" href="css/table.css"/>

Usage

Then in your body section add script with variables "headings" and "rows".

In the array "headings" add as many objects as you want to have headers.
The "label" key is for adding a header name and the "for" key is for assigning values from header to the rows.

In the array "rows" add as many objects as you want to have rows.
Add keys with the same names you used in the "headings" variable.
Don't forget to add all keys in right order even if some of them are empty. 

            
  <script>

        let headings = [
            {
                label: "Nazwa",
                for: "name"
            },
            {
                label: "Ilość",
                for: "quantity"
            },
            {
                label: "Cena",
                for: "price"
            },
            {
                label: "Data",
                for: "date"
            }
        ];

        let rows = [
            {
                name: "Komputer",
                quantity: 1,
                price: "$150",
                date: "09/02/19"
            },
            {
                name: "Monitor",
                quantity: "",
                price: "$100",
                date: "10/18/19"
            },
            {
                name: "Krzesło",
                quantity: 10,
                price: "$20",
                date: ""
            }
        ];
        const table = new Table(headings, rows);   

  </script>
